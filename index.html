Função Quadrática

Uma função quadrática é uma aplicação de R em R que associa a cada número real x o elemento f(x) = ax² + bx + c, onde a ≠ 0 e a, b, c são números reais.

O gráfico da função quadrática é uma parábola.

Exemplos de funções quadráticas:

  • f(x) = x² + 2x + 1
  • f(x) = 2x² - 3x + 5
  • f(x) = -x² + 4x - 7

Características principais:

  • Se a > 0, a parábola é voltada para cima.
  • Se a < 0, a parábola é voltada para baixo.

Função Exponencial

Uma função exponencial é aquela em que a variável aparece no expoente. A base da função deve ser maior que zero e diferente de um.

Forma geral: f(x) = ax, onde a > 0 e a ≠ 1.

Exemplos de funções exponenciais:

  • f(x) = 4x
  • f(x) = (0,1)x
  • f(x) = (2/3)x

Gráfico da função exponencial:

  • Passa sempre pelo ponto (0,1).
  • Não toca no eixo x.
  • A imagem é sempre positiva (não aparece nos quadrantes 3º e 4º).

Sobre o crescimento ou decrescimento:

  • Se a base for maior que 1, a função é crescente (ex: f(x) = 2x).
  • Se a base for entre 0 e 1, a função é decrescente (ex: f(x) = (1/2)x).

Equação do 1º Grau

Uma equação do 1º grau é toda equação que pode ser escrita na forma ax + b = 0, onde a e b são números reais e a ≠ 0.

Exemplos de equações do 1º grau:

  • 2x + 3 = 0
  • -x + 5 = 0
  • 4x - 7 = 0

Como resolver: isolar o x na equação. Exemplo: 2x + 3 = 0 ⇒ 2x = -3 ⇒ x = -3/2.

Equação do 2º Grau

Uma equação do 2º grau é toda equação que pode ser escrita na forma ax² + bx + c = 0, onde a, b e c são números reais e a ≠ 0.

Exemplos de equações do 2º grau:

  • x² + 5x + 6 = 0
  • 2x² - 3x + 1 = 0
  • -x² + 4x - 4 = 0

Fórmula de Bhaskara: x = (-b ± √(b² - 4ac)) / 2a

O discriminante (Δ) é dado por Δ = b² - 4ac. Dependendo do valor de Δ, temos:

  • Δ > 0: duas raízes reais distintas.
  • Δ = 0: uma raiz real dupla.
  • Δ < 0: não existem raízes reais.
